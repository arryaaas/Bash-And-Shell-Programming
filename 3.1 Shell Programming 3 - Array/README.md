# Shell Programming 3

Buatlah program array yang dapat menghitung nilai IPK mahasiswa yang menerapkan beberapa konsep pemrograman bash seperti diatas dengan ketentuan sebagai berikut :
  - user input data arrayIPSMahasiswa[index]

Contoh Output :
```
Input : 3
4
2
3

IPK mhs = 3
```
